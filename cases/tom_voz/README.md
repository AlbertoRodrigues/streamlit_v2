# Case: Tom de Voz

Este case concentra a aplicacao Streamlit usada para avaliar respostas e consolidar notas por modelo.

## Arquivos

- `app.py`: interface principal da aplicacao.
- `respostas.csv`: base local com as respostas avaliadas.

## Como executar

A partir da raiz do projeto:

```bash
streamlit run cases/tom_voz/app.py
```

Se preferir executar de dentro da pasta do case:

```bash
cd cases/tom_voz
streamlit run app.py
```
