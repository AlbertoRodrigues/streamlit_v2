# Case: Investimentos

Este case concentra a aplicacao Streamlit de assistente sobre investimentos com roteamento via LangGraph.

## Arquivos

- `app.py`: interface principal da aplicacao.
- `graph.py`: definicao do grafo e dos agentes.

## Como executar

A partir da raiz do projeto:

```bash
streamlit run cases/investimentos/app.py
```

Se preferir executar de dentro da pasta do case:

```bash
cd cases/investimentos
streamlit run app.py
```
